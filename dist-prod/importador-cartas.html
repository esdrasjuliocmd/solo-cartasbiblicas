Importador de Cartas (Pingão)

Esta página gera um JSON de exemplo e publica cartas em /cartas/<categoria>/popular. Não precisa PowerShell.

1) Configurações
Ex.: https://...workers.dev
Vai publicar em /cartas/{categoria}/popular
2) Gerar arquivo de exemplo

Gera um JSON no formato aceito pelo importador. Você edita e depois importa.

3) Selecionar arquivo para importar

Aceita formatos: { "cartas": [...] }, array direto [...], ou { "personagens": [{ "value": [...] }] }.
